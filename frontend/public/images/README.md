# Footer Images

This folder contains images used in the app (e.g., HRSA logo in the footer).

## HRSA Logo

To download the HRSA logo image, run from the frontend directory:

```bash
npm run download-hrsa-logo
```

Or directly:

```bash
node scripts/download-hrsa-logo.js
```

Source: https://hrsauat2.amer.reisystems.com/LayoutService/images/HealthResourcesandServicesAdministrationlogo.png

The image will be saved as `hrsa-logo.png` in this folder.
